# Changelog

## 0.0.2

- Fixed Windows playback so the extension plays the bundled `assets/yakubu-manage.mp3` sound instead of falling back to a console beep.
- Kept macOS built-in playback unchanged.
- Improved Windows built-in playback handling for both `.mp3` and `.wav` sound files.

## 0.0.1

- Initial release.
- Plays a sound on diagnostic errors and terminal command failures.
- Includes mute toggle, test sound command, and cooldown controls.
